## Features

- User authentication  
- Create and share themed restaurant lists  
- Rate restaurants and leave comments  
- Add friends and view their lists  
- Explore public lists based on different themes  



## Tech Stack

- **Frontend:** React.js
- **Backend / Database:** Supabase 
- **Hosting:** GitHub Pages
